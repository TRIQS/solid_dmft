module reference manual¶

csc_flow contains the charge self-consistency flow control functions
dft_managers DFT code driver modules
dmft_cycle main DMFT cycle, DMFT step, and helper functions
dmft_tools DMFT routine helper functions used during solid_dmft run
postprocessing Postprocessing tools
read_config Provides the read_config function to read the config file
util external helper functions, not used by any DMFT routine
